Co2 Emissions Dashboard

239 words 2 minutes Dashboard Pandas Prophet Time series Plotly Dashboard streamlit

This project explores global fossil CO₂ emissions by country (2002–2022) through interactive dashboards, exploratory data analysis (EDA), and time-series forecasting. The goal is to provide insights into emission trends across different countries and sources, and to predict future emissions using the Prophet forecasting algorithm.

The project demonstrates a mix of dashboarding, visualization, and predictive modeling techniques with a focus on clarity, interactivity, and usability.

Key Highlights

  • Data Exploration & Cleaning: Handled and processed raw emissions data using Pandas.
  • Interactive Dashboard: Built with Streamlit for easy navigation and real-time exploration.
  • Visualizations: Developed rich, interactive plots with Plotly, covering trends by country, region, and emission source.
  • Forecasting: Applied Prophet to predict future CO₂ emissions and visualize potential trajectories.
  • Comparative Analysis: Users can compare countries, regions, or emission sources side by side.

Dataset

  • Source: Global Fossil CO₂ Emissions by Country 2002–2021
  • Size: Covers global emissions data for ~200 countries from 2002 to 2021, broken down by multiple sources of fossil fuel usage.

Tools & Libraries

  • Language: Python 3.x
  • Libraries: Pandas, streamlit, Plotly, Prophet
