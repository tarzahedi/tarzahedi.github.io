Co2 Emissions Dashboard

This project explores global fossil CO₂ emissions by country (2002–2022) through interactive dashboards, exploratory data analysis (EDA), and time-series forecasting. The goal is to provide insights into emission trends across different countries and sources, and to predict future emissions using the Prophet forecasting algorithm.
